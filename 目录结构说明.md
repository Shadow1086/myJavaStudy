# 项目目录结构说明

## 总体结构

```
myJavaStudy/
├── java01/          # 基础阶段：基础语法 + 面向对象
├── java02/          # 进阶阶段：包装类、异常、API、多线程
├── java03/          # GUI 阶段：图形界面编程
├── lib/             # 统一的依赖库（所有子项目共享）
└── docs/            # 项目文档
```

## java01/ - 基础阶段

### 目录说明
- **src/java001/** - 基础语法练习
  - day01-day07：流程控制、循环、数组、类等基础内容
- **Object/** - 面向对象编程
  - Class01-Class11：封装、继承、多态、接口、枚举、注解等
- **Experiment/** - 实验代码
  - Homework01-Homework03：各种实验练习
- **Homework/** - 作业代码
  - 各种作业练习

### 存放规则
- ✅ 所有 .java 源代码文件
- ✅ 学习笔记 .md 文件（与对应章节放在一起）
- ❌ 不存放编译后的 .class 文件
- ❌ 不存放 bin/ 编译输出目录

## java02/ - 进阶阶段

### 目录说明

- **src/Class02/Error/** - 异常处理
- **src/Class03/API/** - Java API 使用

### 存放规则
- ✅ 所有 .java 源代码文件
- ✅ 学习笔记 .md 文件
- ❌ 不存放编译后的 .class 文件
- ❌ 不存放 bin/ 编译输出目录

- ❌ 不存放编译后的 .class 文件

## 根目录

### lib/
- 统一的依赖库目录
- 所有子项目共享此目录中的 jar 文件
- 包含：JUnit、TestNG、Guava 等常用库

### docs/
- 项目总体文档
- README.md - 项目说明
- 目录规划.md - 目录规划文档
- 目录结构说明.md - 本文件

## 文件清理规则

1. ✅ **保留所有 .java 文件** - 源代码必须保留
2. ❌ **删除所有 .class 文件** - 编译输出，应该被忽略
3. ❌ **删除 bin/ 目录** - 编译输出目录
4. ❌ **删除重复的文档文件** - 保留 src/ 中的，删除 bin/ 中的
5. ✅ **保留 .md 文档文件** - 学习笔记和说明文档
6. ✅ **保留 .iml 文件** - IDE 配置文件（虽然 Cursor 不用，但可能有用）

## 注意事项

- 所有编译输出（.class、bin/）都应该被 .gitignore 忽略
- 源代码文件（.java）必须全部保留
- 文档文件（.md）按需保留，删除重复的
