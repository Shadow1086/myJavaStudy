# 项目目录规划

## 目录结构说明

### java01/ - 基础阶段
**存放内容：**
- `basics/` - 基础语法练习（原 src/java001/）
  - day01-day07：流程控制、循环、数组、类等基础内容
- `oop/` - 面向对象编程（原 Object/）
  - Class01-Class11：封装、继承、多态、接口、枚举、注解等
- `exercises/` - 作业和实验（原 Experiment/ 和 Homework/）
  - Homework01-Homework03：各种作业练习
- `docs/` - 学习文档和笔记
  - 各章节的 .md 文档

### java02/ - 进阶阶段
**存放内容：**
- `wrapper/` - 包装类（原 Class01oop03/Wrapper/）
- `exception/` - 异常处理（原 Class02/Error/）
- `api/` - Java API 使用（原 Class03/API/）
- `thread/` - 多线程编程（原 Class03/thread/）
- `docs/` - 学习文档和笔记

### java03/ - GUI 阶段
**存放内容：**
- `awt/` - AWT 图形界面
- `swing/` - Swing 图形界面（未来）
- `javafx/` - JavaFX 图形界面（未来）

### 根目录
- `lib/` - 统一的依赖库（所有子项目共享）
- `docs/` - 项目总体文档（README.md 等）
- `.vscode/` - Cursor/VS Code 配置
- `.gitignore` - Git 忽略规则

## 清理规则

1. **删除所有 .class 文件**（编译输出，应该被忽略）
2. **删除 bin/ 目录**（编译输出目录）
3. **删除重复的 .md 文件**（保留 src/ 中的，删除 bin/ 中的）
4. **保留所有 .java 文件**
5. **保留 .iml 文件**（IDE 配置，虽然 Cursor 不用，但可能有用）
